Awk Exercises
=============
1. calculate the average UID of on the UNIX system that you are on.
	(introductory material. FS, variables, printing).
	use /etc/passwd.
2. find the average group size of groups on the UNIX system that
	you are on.
	(requires split+arrays).
	use /etc/group.
3. find what precentage of your groups on average like to use
	bash as their shell.
	use /etc/passwd and /etc/group.
	use split+arrays.
	use multi file parsing (do everything in one awk run).
